What is a smart contract?

Smart Contracts are contracts written in the form of program code. Smart contracts are compiled into bytecode which is then executed on a blockchain. This bytecode is not easily understood by humans. This tool aims to reverse engineer the bytecode to give users a better understanding of what opcodes are present in a given smart contract.

What are opcodes?

Opcodes are the operations executed on the Ethereum Virtual Machines (EVM). Each opcode represents a unique operation. The full list of opcodes is available in the Ethereum Yellow Paper which is available here.
